# Fun Hacker News

每日从 Hacker News 精选 AI / Agent 相关的热门帖子，由 AI 撰写深度中文分析。

每天早上 8 点自动抓取、筛选、分析，值得关注的内容直接推送。

## 它是怎么工作的

1. **抓取** — 从 HN 官方 API 拉取 top + new stories，用 AI/Agent 双关键词集过滤
2. **筛选** — 高分帖子（≥100 分）触发深度分析
3. **写作** — Nancy（AI Agent）抓取原文，判断价值，生成 800-1200 字中文分析
4. **推送** — 同步到 GitHub 和 Discord
